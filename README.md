# Dr. Ahmadli – Funnel Landing Pages

A 3-page lead generation funnel for **Dr. (Univ. Istanbul) Abbas Ahmadli**, Plastic & Aesthetic Surgery Hannover.

## Pages

| Page | File | Purpose |
|------|------|---------|
| 1 | `index.html` | Opt-in — captures name + email in exchange for free video training |
| 2 | `page2.html` | Video delivery — builds trust, explains the Neue-Ich Gespräch mechanism, drives to booking |
| 3 | `page3.html` | Booking — disclaimer, interactive calendar, confirmation |

## Funnel Flow

```
Instagram / Facebook Ad
        ↓
  Page 1 — Opt-in
        ↓
  Page 2 — Watch Training + Neue-Ich Gespräch
        ↓
  Page 3 — Book the Call
        ↓
  Free 15-min Neue-Ich Gespräch
        ↓
  Full Consultation at Clinic
```

## Design Direction
- **Palette:** Deep plum / ivory / warm gold
- **Typography:** Cormorant Garamond (display) + DM Sans (body)
- **Aesthetic:** Luxury medical editorial
- **Mobile:** Fully responsive

## Deploy on GitHub Pages
1. Upload files to a new repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, root folder
4. Live at `https://yourusername.github.io/repo-name`

---
*Calendar is UI-only — connect to Calendly or equivalent for live bookings.*
